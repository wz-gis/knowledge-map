# Daily Logs

日常日志用于记录当天工程活动、问题、修复和学习，不做复杂任务系统。

## File Name

```text
YYYY-MM-DD.md
```

示例：

```text
2026-05-10.md
```

## Format

```md
# Today
# Problems
# Fixes
# Learnings
# Next Actions
```

## Rules

- 每天一篇即可。
- 不要求完整，重要的是持续。
- 技术问题优先沉淀到 `02_areas/` 或 `snippets/`。
- 项目相关内容定期移动到 `01_projects/`。
